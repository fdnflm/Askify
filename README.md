## Описание

Сервис, позволяюший пользователям задавать пользователям вопросы, в том числе анонимно.

## Канбан доска проекта

[![Notion](https://img.shields.io/badge/-Notion-black?logo=notion&logoColor=white)](https://www.notion.so/fdnflm/56611a08bce14df49fe1ee4edee128a0?v=666fde18131c4e2fab693a483e6c0a1d)

## Функционал

* Лента вопросов
* Подписки
* Информация о пользователе
  1. Имя, фамилия
  2. Местоположение
  3. Instagram
  4. Telegram
  5. Bio
* Настройки
* Вопросы
  1. Удаление
  2. Ответ
  3. Отправка
      1. Анонимно
      2. От пользователя
